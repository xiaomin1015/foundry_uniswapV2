# foundry_uniswapV2
implement uniswap V2 in foundry framework

###  solidity version
using solidity version: 0.8.17

###  build
```
        forge build
```       

### Run the test:
```
        forge test
```
for more test options, check this <a href="https://book.getfoundry.sh/forge/tests" class="internal-link" target="_blank">tutorial</a>.
### see test coverage
```
        forge coverage
```
